# 📚 guide_for_some_attacks

A small collection of **practical examples and walkthroughs** for some common attacks and techniques, mainly for **pentesting and learning purposes**.

## 🔍 What this repo is about

This repository gathers simple, self‑contained examples that demonstrate how certain vulnerabilities, client‑side weaknesses, server-side weakness, misconfig, ... can be observed and exploited in controlled environments.  
It is **not** a tool suite, but more of a **hands‑on cheatsheet** you can use to:

- Understand the basics of some attacks.
- Reproduce simple scenarios in a lab.
- Use as a reference or demo for reports or workshops.

## ⚠️ Usage and ethics

- This guide is **for educational and authorized pentesting only**.  
- Do **not** use these examples against systems you don’t own or without explicit permission.  
- Always respect the law and your clients’ scope.
